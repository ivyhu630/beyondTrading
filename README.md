## Getting Started

1. Download dependencies with:

   > `npm install`

2.  Set up `.env` file using `/example.env`:

    > `.env` requires github API key and a local PORT

3. Run webpack in development with:

   > `npm run client-dev`

4. Run server with:

   > `npm run server-dev`
